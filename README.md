# Dishonored Fan Wiki & Community Hub

A responsive multi-page website dedicated to the "Dishonored" game universe. This project was built to demonstrate front-end development skills, including responsive layout design, UI/UX styling, and interactive JavaScript functionality.

## Live Demo
[View the Live Project here](GITHUB_PAGES)

## Tech Stack
* **HTML5 & CSS3** – Semantic structure and custom styling.
* **Bootstrap 5** – Responsive grid system and UI components (Navbar, Cards, Buttons).
* **JavaScript (ES6)** – Interactive features and form logic.
* **Google Fonts** – Integrated typography for a thematic "steampunk" atmosphere.

## Key Features
* **Interactive Voting System:** A custom-built poll where users can vote for their favorite game, with dynamic progress bar updates via JavaScript.
* **Smart Registration Form:** Includes real-time client-side validation for:
    * Email format verification.
    * Password strength and matching confirmation.
* **Game Gallery:** A dedicated section displaying high-quality game assets with a clean, centered layout.
* **Fully Responsive:** Optimized for desktops, tablets, and mobile devices using Bootstrap's flexible grid.
* **Community Section:** A mock-up of a forum interface with upvote/view counters to simulate a real user environment.

## Project Structure
* `index.html` – Main landing page with game info and system requirements.
* `game.html` – Page about game.
* `content.html` – Community forum and voting interface.
* `forum-register.html` – Interactive user sign-up page.
* `redirect.html` – Redirect after registration.
* `/css` – Custom stylesheets and Bootstrap overrides.
* `/js` – Form validation and voting scripts.
* `/img` – Game assets and background textures.
